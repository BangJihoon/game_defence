# Shop System Specification
Version: 1.0 (Updated 2026-03-08)

---

## 1. 상단 공통 재화 바 (Top Currency Bar)
모든 페이지 상단에 고정되며, 바로 구매가 가능한 '+' 버튼이 포함된다.
- **전투 에너지 (Energy)**: 스테이지 진입 재화.
- **보석 (Gems)**: 유료 재화. 캐릭터 카드 및 에너지 구매에 사용.
- **코인 (Coins)**: 게임 내 재화. 캐릭터 강화 및 신전 업그레이드에 사용.

### 1.1 재화 구매 비율 (Quick Purchase)
- **에너지**: 25 에너지 = 500 보석 (일일 3회 한정) / 광고 시청 시 5 에너지 획득.
- **보석**: 500개 = $1 (3,000원) / 3,000개 = 10,000원.
- **코인**: 10만개 = $1 (3,000원) / 50만개 = 10,000원.

## 2. 오늘의 제안 (Today's Offers - 상점 상단)

### 2.1 오늘의 할인 (Rotation Deal)
- **로직**: 10가지 상품 리스트를 10일 주기로 순환하며 매일 1개씩 노출.
- **가격**: VIP 혜택가보다 높은 일반 할인가 적용.
- **리스트**: 재화(1k~100k) 및 등급별 카드(실버 10, 골드 3, 플레 1, 다이아 1).

### 2.2 구독 상품 (Subscription)
- **월간 VIP (Monthly)**: 4,900원. 광고 제거, 보상 2배, 매일 2,000보석.
- **연간 VIP (Yearly)**: 49,000원. 광고 제거, 보상 2배, 매일 3,000보석. (VIP Zone 접근 권한)

### 2.3 VIP Zone (Special Access)
- **매일 보상**: 캐릭터 조각 (실버 5, 골드 1), 룬, 특정 재화 무료 획득.
- **전용 팩**: 
  - 보석 1만: 30,000원
  - 보석 5만: 99,000원
  - 보석 10만: 198,000원

## 3. 캐릭터 탭 (Character Tab - 상점 하단)

### 3.1 무료 보상 (Ad Reward)
- 10분마다 광고 시청 후 실버 카드 1~3장 획득 가능.

### 3.2 등급별 카드 가격 (Gems)
- **실버**: 100 보석
- **골드**: 500 보석
- **플래티넘**: 2,500 보석
- **다이아몬드**: 구매 불가 (전용 보상 및 승급으로만 획득)

### 3.3 가치 치환 공식 (Conversion Logic)
- **실버 50장 = 골드 5장 = 플래티넘 1장**
- 시스템적으로 하위 카드를 상위 카드로 치환하거나, 동일 가치의 재화로 환산하여 관리 가능.

## 4. 데이터 관리 방식
- 모든 상품 정보는 `assets/data/shop_data.json`에서 관리한다.
- `PlayerDataManager`는 마지막 구매 시간, 일일 횟수 제한, VIP 상태 등을 추적한다.
